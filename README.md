# Sentra UMKM

Sentra UMKM adalah prototype website berbasis web yang dirancang sebagai platform digital untuk membantu pengelolaan Usaha Mikro, Kecil, dan Menengah (UMKM). Aplikasi ini menyediakan simulasi sistem administrasi yang memudahkan pengguna dalam mengelola data usaha, produk, transaksi, serta informasi UMKM melalui antarmuka yang sederhana dan responsif.
